# موقع تعريفي ومنصة فعاليات - مبادرة قِمّة

## مقدمة
مشروع **موقع تعريفي ومنصة لتجميع الفعاليات (هاكاثونات، مؤتمرات، معارض، ودورات)**، يهدف إلى توحيد عرض الفعاليات التقنية والتعليمية في مكان واحد تابع لمبادرة **قِمّة**.

يُتيح الموقع للزوّار استعراض الفعاليات القادمة والبرامج الجارية، بينما يوفّر للمسؤولين لوحة تحكم لإدارة المحتوى بسهولة.

## الهدف
> إنشاء موقع تعريفي حديث وسهل الاستخدام يعرض:
- تعريف مبادرة قمة وبرامجها.
- تقويم فعاليات تفاعلي مع خاصية البحث والفلترة.
- صفحة تفاصيل لكل فعالية وروابط التسجيل.
- لوحة تحكم للمسؤولين لإدارة المحتوى والفعاليات.


  ## نطاق العمل
- صفحات تعريف (الرئيسية، عن المبادرة، البرامج).  
- تقويم فعاليات + تفاصيل الفعالية.  
- لوحة تحكم (CMS) لإدارة الفعاليات والمحتوى.  
- ربط روابط التواصل الاجتماعي للمبادرة.  
- دعم اللغة العربية (مع إمكانية دعم الإنجليزية مستقبلًا).


  ## المستخدمون المستهدفون
| الفئة | الوصف |
|-------|--------|
| الزوار | طلاب، مبرمجون، باحثون عن فرص |
| المحررون | مسؤولون عن إضافة وتحديث الفعاليات |
| المدير | يعتمد المحتوى ويدير النظام |

## التقنيات المستخدمة
-والفعاليات.

-Node.js / Express  
- Database: MySQL  
- Authentication: JWT  
- Frontend: HTML, CSS, JavaScript  
- Environment: dotenv  
- Version Control: Git & GitHub
